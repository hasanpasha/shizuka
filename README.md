
# stream-cli
## a command line tool to watch movies and series

## Usage
first, download the source code:
```
git clone https://github.com/hasanpasha/stream-cli/
```
```
cd stream-cli/
```
Now you need to get the reuqired packages:
```
pip install -r requirments.txt
```
To run the app:
```
python main.py
```
